## Project Overview
- Dataset: [MNIST](http://yann.lecun.com/exdb/mnist/) (60,000 training images, 10,000 test images).
- Framework: TensorFlow / Keras.
- Model: CNN with 2 convolution layers + dense layers.
- Accuracy: ~98% on test data.
